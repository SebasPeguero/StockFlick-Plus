# StockFlick+ 
Sistema de Gestión de Inventario desarrollado para la materia de Programación II en el ITLA.

## Características
- Gestión completa (CRUD): Crear, Leer, Editar y Eliminar productos.
- Persistencia de datos en formato JSON.
- Buscador en tiempo real con JavaScript.
- Manejo de errores y decoradores personalizados.

## Cómo ejecutar
1. Clona el repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta el servidor: `uvicorn app:app --reload`
